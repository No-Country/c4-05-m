# Bienvenid@s a Fashion Hunter

## ¿Que es?
Es una red social de moda, donde podés ver y guardar Outfits de tus contactos, subir tus posteos y someterlos a votación.

## ¿Que tecnologías usamos?

- Typescript
- Angular 13
- Html
- Css
- Node Js
- MongoDB
- Mongoose
- Express
